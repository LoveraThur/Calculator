Este arquivo é uma calculadora simples feita em python, com coisas que aprendi no curso básico de python, como módulos, funções, bibliotecas, entre outros conceitos do Python básico.
Com o foco máximo em praticar a lógica e os conceitos estudados, utilizando o minimo possivel de bibliotecas(apenas 1 operação, para mostrar que eu sei um pouco sobre bibliotecas Python)

esta calculadora contêm:

  inputs de numeros REAIS(com ponto(.), que é a leitura padrão do Python)
  
  escolha de operação, contendo:
  
    adição
    
    subtração
    
    multiplicação
    
    divisao
    
    fatorial
    
    raiz quadrada
    
    cubo

    Porcentagem (acréscimo e desconto)
